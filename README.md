## js-commonFunction
工作中常见的一些js通用函数。主要分为数组，正则，对象，ajax，cookie，随机，其他七部分。
#### 数组部分
  - 判断是否为数组
  - 数组去重
  - 复制数组
  - 数组最大值
  - 数组最小值
  - 类数组转为对象
#### 正则部分
  - 邮箱正则
  - 手机正则
  - 中文正则
#### 对象部分
  - 对象的深克隆
#### ajax
  - 常见的get，post，delete，put请求封装
#### cookie
  - 获取指定cookie值
  - 添加cookie值
#### 随机部分
  - 随机颜色
  - 指定长度的随机字符串
  - 指定范围的随机数字
  - 指定范围的随机正数
#### 其他部分
  - 回车事件的绑定
  - 以x为底y的对数
